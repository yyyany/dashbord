# 📊 Dashboard Personnel

Un dashboard personnel web servant de centre de contrôle quotidien (tâches, météo, système, calendrier, notes, etc.).

## 🧱 Structure du Projet

```
📁 dashboard-project
├── frontend/                # React + Tailwind
│   ├── components/
│   ├── pages/
│   ├── utils/
│   └── App.jsx
├── backend/                 # Express + MongoDB
│   ├── routes/
│   ├── controllers/
│   ├── models/
│   └── server.js
├── nginx/                   # Conf nginx et certbot
├── scripts/                 # Scripts CPU/RAM
├── PROGRESS.md              # Suivi d'avancement
└── README.md                # Documentation
```

## 🚀 Installation & Démarrage

### Prérequis
- Node.js (v16+)
- MongoDB
- Git

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm start
```

## 🧩 Fonctionnalités
- Tâches & objectifs
- Météo en temps réel
- Monitoring système
- Intégration calendrier
- Bloc-notes markdown
